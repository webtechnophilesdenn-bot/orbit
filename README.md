# Orbit

## Prerequisites

- Node.js (v18+)
- PostgreSQL running locally

## Setup

### 1. Backend

```bash
cd Backend
cp .env.example .env   # then edit .env with your database credentials
npm install
npm run dev            # starts on http://localhost:5000
```

### 2. Frontend

```bash
cd Frontend
npm install
npm run dev            # starts on http://localhost:5173
```
